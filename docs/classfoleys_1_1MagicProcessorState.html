Detailed Description

The MagicProcessorState is the docking station for the MagicPluginEditor.

It grants access to the processor itself and the AudioProcessorValueTreeState. It is also the place, where the data for the visualisers is sent to, which are MagicPlotSources and MagicLevelSources.
